What happens with the eigenvalues if you compute eig(A-a*eye(2))

Try with a=1, 1i, 0.5+0.5i

I=eye(2);
eig(A-I)
eig(A-1i*I)
eig(A-(0.5+0.5i)*I)

  -1.1000 + 1.0000i
  -1.1000 - 1.0000i
